# slackgpt


## Slack Configuration

### Setting Socket Mode

- Enable Socket Mode
- Enable Events in Event Subscriptions
  - Add app_mention in Subscribe to bot events
  

### Scopes in OAuth & Permissions

- app_mentions:read
- chat:write
- im:write



## ENVIRONMENT

```
OPENAI_API_KEY=
SLACK_SIGNING_SECRET=
SLACK_BOT_TOKEN=xoxb-
SLACK_APP_TOKEN=xapp-
```


## Run server

```
npm i
node index.js

```
